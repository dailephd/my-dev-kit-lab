What was delivered

This batch delivered the complete raw-vs-indexed experiment pipeline for my-dev-kit-lab,
including real-agent campaign support for Codex and Claude.

Implemented in this batch:

- Benchmark projects at small, medium, and large complexity levels
- Project complexity metrics and benchmark case metadata with answer keys
- Prompt variant generation at short, medium, long, and multi-step complexity levels
- Prompt complexity metrics
- fake-agent adapter (deterministic, no external CLI required)
- Codex adapter (requires local Codex CLI)
- Claude adapter (requires local Claude CLI)
- Windows-safe CLI command resolution
- Controlled experiment runner with strategy pairing and correctness scoring
- Token usage, duration, and status comparisons between matched strategy pairs
- HTML experiment report rendering
- Static SVG plot generation
- Optional PNG screenshot capture via Playwright
- Gallery manifest and static gallery index
- Visualization demos using my-dev-kit commands
- Final demo workflow (single command, full pipeline)
- Real-agent campaign support with structured outcomes
- Metric glossary (docs/METRICS.md)

Real-agent campaign results

A real-agent campaign was run against the medium and large benchmark projects using
Codex and Claude. Results were partial. See docs/REAL_AGENT_BENCHMARK_CAMPAIGN_REPORT.txt
for the full campaign report.

- Codex produced some timeouts and invalid-output runs
