---

Overview

This report summarizes the real-agent benchmark campaign run against the medium and large
benchmark projects using Codex and Claude. The campaign infrastructure ran end-to-end and
recorded all outcomes as structured JSON. Results are partial because real-agent CLIs can
produce completed runs, timeouts, invalid output, and session limits.

These results demonstrate that the experiment infrastructure works with real agents. They
do not constitute proof of token savings. Stronger evidence requires future experiment
types such as warm-index reuse, incremental-change, and context-window scaling.

See docs/ROADMAP.md for the planned next phases.

Known limitations

- Claude does not expose token totals. Token savings comparisons are unavailable for
  Claude runs. The report shows N/A for token savings on all Claude run pairs.

- Codex may expose token totals but produced some timeouts and invalid-output runs
  during this campaign. Token savings are only shown for completed Codex run pairs
  where both runs returned token totals.

- Campaign results are partial. Not all runs completed. The report shows warnings for
  missing token totals and non-completed runs.

- The current baseline does not prove token savings. It demonstrates that the experiment
  infrastructure works end-to-end with real agents and records structured outcomes.

---

Outcome types recorded

- completed:      Agent returned a valid response
- timeout:        Run exceeded the timeout limit
- invalid-output: Agent returned output that could not be parsed
- limit-reached:  Agent hit a usage or session limit
